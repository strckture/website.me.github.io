Tracking

„I spy with my little eye.“
This whole idea was triggered by the movie Mission: Impossible - Ghost Protocol. (reference)
The hallway scene really sparked my interest in recreating a perspective tracking program. I started to get to this concept by trying out different types of tracking. First using only Processing and later with the OpenCV library. At the moment the prototype tracks a face via the webcam of a computer and displays a 3D model in a 3D scenario.
Furthermore I started thinking about the applications for these types of technologies. So maybe if you want to view an item a little more closely while shopping online this might be a way to do it.

Circle Font

Circle font is not just only a pretty round font.
The idea for the font was developed in a lecture. In that lecture I learned that all uppercase letters can be reduced to either a square, a triangle or a circle. So I thought in the sense of minimalism it would be interesting to try to reduce these shapes to only one per font (for all the letters).
I decided to use circles because I found it to be the smoothest looking shape and since I like space and planets it is also the best one to fit that theme.
The result is a thin font which is used best in short headlines. In the end it is still a pretty round font and thatâ€™s why it is called Circle Font.
